## 📖 LISEZ-MOI - Ortho4XP_V2.0 🌍

**Évolution moderne et installation automatique sans terminal d’Ortho4XP_V2.0 par Ypsos sur la base d'ORTHO4XP_MASTER_1.40. issue de la version du créateur d'ORTHO4XP : Oscar Pilote**

---

## 🚀 Pourquoi cette version V2.0 ?

L'objectif de ce projet est de lever la barrière technique du terminal. Cette version simplifie radicalement l'expérience utilisateur tout en conservant la puissance de l'outil original d'Oscar Pilote.

### ✨ Les points forts :
* **📦 Zéro Terminal :** Installation et lancement automatisés.
* **🖱️ Accessibilité :** Conçu pour les simmers qui veulent se concentrer sur le vol, pas sur le code.
* **🛠️ Fiabilité :** Reprise de la base solide 1.40 Master avec des optimisations modernes.

---

## 🛠 Utilisation Rapide

1.  **Consultez** impérativement le fichier [AVERTISSEMENT_LICENCE_LEGAL.md](./AVERTISSEMENT_LICENCE_LEGAL.md).
2.  **Téléchargez** le dépôt.
3.  **Lancez** l'exécutable/script d'installation automatique.
